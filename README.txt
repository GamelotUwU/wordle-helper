# Wordle Helper

A console application written in Python that assists in solving Wordle puzzles.

## Features

- Filter by green letters
- Filter by yellow letters
- Filter by gray letters 
- Reads a word list from a text file
- Writes remaining possible solutions to a file
- Reset function for result file and hints(currently bugged, if used program needs to be run again otherwise result list won't update)

## Technologies

- Python
- File I/O
- Functions 
- Lists
- Tuples
